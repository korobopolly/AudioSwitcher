본 프로젝트는 Windows 환경에서 자주 사용하는 오디오 장치를 보다 간편하게 전환할 수 있도록 지원하는 경량 유틸리티입니다.

우측 하단 작업표시줄 트레이에 상주하며, 사용자가 미리 등록한 즐겨찾기 장치 2개를 트레이 아이콘 클릭만으로 순차적으로 전환할 수 있습니다.
기존의 키보드 단축키(Ctrl + Win + V) 사용 없이도 빠르고 직관적인 장치 전환을 제공합니다.
  
  시작 프로그램에 등록하려면:
  1. Win + R → shell:startup 입력
  2. AudioSwitcher.exe의 바로가기를 해당 폴더에 넣기
